Marin Companion is a companion plugin for Marin theme.

== Description ==

This is companion plugin is created for Marin theme.
The plugin add custom frontpage sections for Marin theme and it will allow you full customizer settings for the theme front page sections. 
The plugin options are user-friendly and you can use this plugin for all marin child themes also.
